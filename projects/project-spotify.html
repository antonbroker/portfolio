Spotify with AI

This project is a frontend music streaming application inspired by Spotify, enhanced with AI-driven recommendations. It allows users to search for songs, create playlists, and receive personalized music suggestions powered by machine learning.

React — building a responsive and interactive UI.
JavaScript (ES6+) — client-side logic and API integration.
REST API — fetching music data and metadata.
AI algorithms — generating playlist recommendations.

Key Features:
• Modern Spotify-like interface for browsing and playing music
• AI-powered playlist and song recommendations
• Search functionality with real-time API calls
• Playlist creation and management by the user
