Portfolio Website

This project is a modern portfolio website built to showcase my skills, projects, and experience. It features a responsive layout, interactive sections, and advanced animations to deliver a dynamic user experience.

HTML5 & CSS3 — semantic structure and responsive design.
JavaScript (ES6+) — interactivity, theme switching, smooth scrolling.
GSAP & CSS animations — rich transitions and visual effects.
GitHub Pages — deployment and hosting.

Key Features:
• Fully responsive design (desktop & mobile)
• Dark/light/cyberpunk theme switching
• Animated sections with scroll effects
• Integrated links to GitHub, LinkedIn, and contact form
