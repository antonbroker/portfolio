e-Commerce Platform with AI

This project is a full-stack e-commerce application with an integrated AI recommendation engine. It allows users to browse products, add them to a shopping cart, proceed with checkout, and get personalized product suggestions.

Node.js & Express.js — backend logic and RESTful API.
MongoDB — storing products, users, and orders.
React — building a dynamic and user-friendly storefront.
AI algorithms — product recommendations and user personalization.

Key Features:
• Product catalog with categories and search
• Shopping cart and checkout process
• Authentication and user accounts
• AI-driven recommendations to boost user engagement
• Secure payment and order history (planned)
